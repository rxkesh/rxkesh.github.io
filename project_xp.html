Pokécounter

Pokécounter is a django-based website. The objective of the site is to provide users with counters for a given pokémon. I created this site for personal use and to learn how to use django. This project taught me how to use django template filters, the basics of CSS, HTML, and Javacsript, the basics of Python, and more.
